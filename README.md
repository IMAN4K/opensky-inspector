# opensky-inspector
Inspector and visualizer for opensky-network.org state vector samples archive

# Get started
1.Run main.py
    `python3 main.py`
2.After prompt of interactive console:
    `> load ~/opensky-inspector.ini`
    `> download 2020-05-11 2021-05-11`
    `> import`
    `> visualize 2020-05-11 11:55:31`
